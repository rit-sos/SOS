NAME
	windowing_draw_line

SYNOPSIS
	void windowing_draw_line(Uint x0, Uint y0, Uint x1, Uint y1, Uint8 r, Uint8 g, Uint8 b)

DESCRIPTION
	Draws a line in the user buffer from (x0, y0) to (x1, y1) using the
	specified color.
	
	If part of the line lies off the screen, it is not drawn.

RETURN VALUE
	None
